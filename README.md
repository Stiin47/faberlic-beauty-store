# Магазин косметики Faberlic

Сайт представителя компании Faberlic, выполненный в формате интернет-магазина косметики и товаров для дома.

Демо: [https://stilin47.github.io/faberlic-beauty-store/](https://stilin47.github.io/faberlic-beauty-store/)

## О проекте
Сайт предназначен для ознакомления с брендом Faberlic, его продукцией и преимуществами работы представителей компании. Проект демонстрирует ассортимент товаров и модели интернет-магазина.

## Функциональность сайта
- Информация о компании Faberlic
- Каталог косметики и товаров
- Описание преимуществ сотрудничества
- Простая и удобная навигация

## Используемые технологии
- HTML5
- CSS3
- JavaScript
- GitHub Pages (для хостинга)

## Запуск проекта локально
1. Скачайте репозиторий
2. Откройте файл index.html в браузере

## Статус проекта
Проект находится в разработке и может улучшаться.
